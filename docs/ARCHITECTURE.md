# LAEX Architecture

## Filosofía

LAEX no se desarrolla por pantallas.

LAEX se desarrolla por módulos.

Cada módulo debe cumplir cinco reglas:

1. Ser reutilizable.
2. Ser escalable.
3. Tener una única responsabilidad.
4. No duplicar información.
5. Mantener la identidad visual de LAEX.

---

## Orden de desarrollo

1. Core
2. Identity
3. Design System
4. Components
5. Modules
6. AI
7. Blockchain
8. Apps
